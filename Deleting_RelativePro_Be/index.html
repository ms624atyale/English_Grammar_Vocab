The items ordered online
✓
Noun + Past Participle (p.p.)
Noun + [which/that were] + p.p.
인터넷으로 주문된 물품들 (수동 관계)
The items ordered online will arrive tomorrow. ※ 원래 문장: The items [which were] ordered...
The manager leading the project
✓
Noun + Progressive (-ing)
Noun + [who is/was] + -ing
프로젝트를 이끄는 팀장 (능동/진행 관계)
The manager leading the project is highly experienced. ※ 원래 문장: The manager [who is] leading...
Documents vital for the audit
✓
Noun + Adjective Phrase
Noun + [which are] + Adjective

Please submit all documents vital for the financial audit. ※ 원래 문장: ...documents [which are] vital...
